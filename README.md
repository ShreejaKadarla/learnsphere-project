# LearnSphere - Hackathon Project

LearnSphere is an AI-powered educational platform designed to simplify learning by providing interactive tools, personalized recommendations, and performance tracking. This hackathon project demonstrates a smart, user-friendly system to enhance learning efficiency and engagement for students.

## Features
- Personalized learning recommendations
- Interactive tools for practice and quizzes
- Simple, user-friendly interface
- Performance tracking and feedback

## Project Structure
- `src/` - Python scripts for core functionality
- `assets/` - Images, datasets, and other resources
- `README.md` - Project documentation

## How to Run
1. Extract the ZIP file.
2. Navigate to the project folder.
3. Run the main Python file (e.g., `python main.py`) to start the application.

## Technologies Used
- Python
- AI/ML libraries (e.g., scikit-learn, TensorFlow)
- Streamlit for web interface
